Colecciones

Listas

Una lista puede contener varios elementos

["manzana", "platano", "naranja"]

Están ordenadas y empiezan en el índice 0

Se pueden cambiar, añadir y borrar elementos de la lista

Permite elementos duplicados

Para saber el tamaño de una lista usamos la funcion len(nombreLista)

Se pueden mezclar tipos de datos en los elementos de la lista

La funcion type() devuelve class "list"

Tuplas

Puede contener varios elementos

("manzana", "platano", "naranja")

Están ordenadas y empiezan en el índice 0

No se pueden modificiar, añadir o borrar elementos

Permite duplicados

Para saber el tamaño de una lista usamos la funcion len(nombreLista)

Se pueden mezclar tipos de datos en los elementos de la lista

La funcion type() devuelve class "tuple"

Tiene que tener al menos una coma y un elemento

Para crear una vacia nomTupla=tuple()

Diccionarios

Almacen de informacion en formato clave:valor

{"clave1":"valor1","clave2":"valor2"}

Están ordenadas

Se pueden modificiar, añadir o borrar elementos

No permite claves duplicadas

Se pueden mezclar tipos de datos en los valores de la lista pero las claves son strings

La funcion type() devuelve class "dict"

Conjuntos

Puede contener varios elementos

{"manzana", "platano", "naranja"}

No están ordenadas el 0 puede ser cualquier cosa

No se pueden modificiar elementos pero si añadir o borrar elementos

No permite duplicados

True y 1, False y 0 son lo mismo por lo que lo cuenta como duplicado

Se pueden mezclar tipos de datos en los elementos de la lista

La funcion type() devuelve class "set"

Para saber el tamaño de una lista usamos la funcion len(nombreLista)
